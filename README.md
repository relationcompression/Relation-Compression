# Relation Compression (RC): A New Lens

**Relation Compression (RC)** is a proposed framework that explores the hypothesis that intelligence emerges through the compression, organization, and activation of relational structures rather than the storage of isolated facts.

Instead of asking only *what* an AI knows, RC asks a different question:

> **How are relationships represented, compressed, activated, and generalized inside an intelligent system?**

This repository contains the first public draft of the **Relation Compression** framework and introduces a set of experimental methodologies called **Model Relation MRIs (MRIs)**. These experiments investigate relational representations across multiple language models and examine how relational organization may evolve with model scale.

## Vision

RC aims to provide a new perspective for understanding intelligence in both artificial and biological systems. Rather than treating intelligence as memorization or parameter count alone, RC explores whether the ability to efficiently compress and manipulate relational knowledge forms a fundamental component of reasoning, abstraction, and generalization.

## Current Status

This repository represents **Version 1** of the Relation Compression framework.

RC should be viewed as an evolving research program rather than a finished theory. Future work will include additional MRI experiments, stronger validation, mathematical formalization, larger model evaluations, and independent replication.

## Research Philosophy

Scientific progress depends on evidence, replication, criticism, and continuous refinement.

This work is presented as an invitation to investigate, challenge, improve, and extend the Relation Compression framework through rigorous experimentation.
